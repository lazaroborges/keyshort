# Quick Start Guide

## Install the Extension (2 minutes)

1. Open Chrome and go to `chrome://extensions/`
2. Turn ON "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Navigate to and select this folder: `/Users/lazaro/Documents/javascript/keyshort/`
5. Done! The extension is now installed

## Test It Immediately

1. Open `test.html` (in this folder) in Chrome
2. Click in any text field
3. Type: `lazaro-`
4. Wait half a second
5. Watch it change to: `lazaro@parsa-ai.com.br`
